Material-3 Colors API

Important: Colors are hexadecimal CSS colors without the number sign (#)! If you request a preprocessor endpoint all values are lists and not wrapped by color funtions.

Available endpoints

/theme.{json, scss, sass, less, stylus}

Creates a Material 3 Theme base on one color.

Query parameter

  • color: <css-color> (required)
  • name?: <string> (only for JSON output)
  • custom?: <name>?:<css-color> (multiple)
  • prefix?: <string>
  • target?: hex | hsl | rgb | rgba
