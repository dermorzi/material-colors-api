Material-3 Colors API

Important: Colors are hexadecimal CSS colors without the number sign (#)!

Available endpoints

/theme

Creates a Material 3 Theme base on one color.

Query parameter

  • color: [color]
  • custom: [color] | [name:color] | Array<[name:color]> (optional)

/tonal

Creates a Material 3 Color Palette based on the color.

Query parameter

  • color: [color]

/neutral

Creates a Material 3 Surface Color Palette based on the color with a chroma of 10%.

If no color is provided the palette is based on white.

Query parameter

  • color: [color] (optional)
